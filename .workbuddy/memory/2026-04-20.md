# 2026-04-20 工作日志

## MosMenu 折叠箭头修复
- **问题1**：`anchors.top: captionbar.top` 导致 menu 从窗口顶部开始被标题栏遮挡 → 改为 `captionbar.bottom`
- **问题2**：`MosMenu.qml` 的 `__expandedIcon`（`MosIconText`）`iconSource` 被注释掉（因为 `MosIcon` 枚举为空），导致展开箭头不显示
  - 根因：`MosIcon.h` 的枚举类完全为空，`MosIcon.DownOutlined` = undefined
  - 修复：将 `__expandedIcon` 从 `MosIconText` 改为标准 `Text`，使用 Unicode `\u25BE`（▾），配合 `rotation` + `Behavior` 做平面旋转动画
- **问题3**：`menuButton.expandedVisible` 属性不存在（未声明）→ 改为 `menuButton.showExpanded`
